# History Archive — 역사로 읽는 오늘의 시장

역사라는 데이터로 오늘의 시장과 나를 비춰보는 **학습·참고용 플랫폼**입니다.
돈의 역사 타임라인, 사건별 심층 아카이브, 시장 연계 차트, 그리고 매일의 선택이 쌓이는 "갈림길"로 구성됩니다.

> **중요 — 이 서비스는 투자 자문이나 권유가 아닙니다.**
> 모든 콘텐츠는 역사적 사실에 기반한 학습·참고용 자료이며, 미래의 결과를 예측하거나 보장하지 않습니다.
> 모든 투자 판단과 책임은 이용자 본인에게 있습니다.

---

## 빠른 시작

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행 (http://localhost:5173)
npm run dev

# 3. 프로덕션 빌드 (dist/ 폴더 생성)
npm run build

# 4. 빌드 결과 미리보기
npm run preview
```

Node.js 18 이상을 권장합니다.

---

## 기술 스택

- **React 18** — UI
- **Vite 5** — 빌드 도구 (정적 사이트 출력)
- **Framer Motion** — 애니메이션
- **lucide-react** — 아이콘
- **Pretendard** — 폰트 (런타임에 CDN으로 주입)

의존성이 가볍고, 빌드 결과는 순수 정적 파일(HTML·CSS·JS)이라 어떤 정적 호스팅에도 올릴 수 있습니다.

## 배포

`npm run build` 후 생성되는 `dist/` 폴더를 그대로 정적 호스팅에 올리면 됩니다.

### Vercel
```bash
npm i -g vercel
vercel
```
프레임워크 자동 감지(Vite). 빌드 명령 `npm run build`, 출력 디렉토리 `dist`.

### Netlify
- 빌드 명령: `npm run build`
- 게시 디렉토리: `dist`

### Cloudflare Pages
- 빌드 명령: `npm run build`
- 빌드 출력 디렉토리: `dist`

### GitHub Pages (서브경로 배포 시 주의)
저장소 하위 경로(`https://id.github.io/repo/`)에 올릴 경우,
`vite.config.js`의 `base`를 `"/repo/"`로 바꾼 뒤 빌드하세요.
루트 도메인 배포면 `base: "/"` 그대로 둡니다.

> 이 앱은 라우터 없이 내부 `view` 상태로 화면을 전환하는 단일 페이지(SPA)입니다.
> 별도의 SPA 리다이렉트 설정 없이도 동작하지만, 호스팅에 따라 404 폴백을 `index.html`로 지정하면 더 안전합니다.

## 프로젝트 구조

```
history-archive/
├─ index.html          # 진입 HTML (메타·OG 태그 포함)
├─ package.json
├─ vite.config.js      # base 경로 등 빌드 설정
├─ src/
│  ├─ main.jsx         # React 진입점
│  ├─ index.css        # 전역 리셋
│  ├─ App.jsx          # 앱 루트 — 라우팅(view 상태)과 갈림길 상태 관리만
│  ├─ data/            # 모든 데이터 (코드 수정 없이 여기만 고치면 됨)
│  │  ├─ theme.js      #   디자인 토큰 (색·폰트·navBtn)
│  │  ├─ icons.js      #   아이콘 매핑 (ICON_MAP)
│  │  ├─ i18n.js       #   다국어 사전 (L, I18N) ← 번역 수정은 여기
│  │  ├─ events.js     #   갈림길 사건, 등급, 투자 초상 유형
│  │  ├─ timeline.js   #   6시대, 세부 사건, 사건 심층 상세, ERA_ICON
│  │  ├─ market.js     #   시세·기간 차트·변동 이슈, 역사 사례, 파트너
│  │  └─ mirror.js     #   날짜별 거울 데이터 (현재 미사용, 보존)
│  └─ components/      # 화면별 컴포넌트
│     ├─ Header.jsx        # 상단 네비 + 언어 전환
│     ├─ Timeline.jsx      # 타임라인(홈)
│     ├─ EventDetail.jsx   # 사건 상세 페이지
│     ├─ EventChart.jsx    #   상세용 가격 곡선 차트
│     ├─ MarketDemo.jsx    # 시장 연계 화면
│     ├─ QuoteChart.jsx    #   기간 전환 시세 차트
│     ├─ MyPage.jsx        # 마이페이지
│     ├─ Fork.jsx          # 갈림길 화면
│     ├─ ForkPopup.jsx     # 갈림길 로그인 팝업
│     ├─ AdSlot.jsx        # 광고 슬롯
│     ├─ Disclaimer.jsx    # 면책 고지
│     └─ _unused/          # 현재 라우팅에서 빠진 컴포넌트 보관 (Home, Mirror 등)
└─ dist/               # 빌드 결과물 (배포 대상)
```

**모듈화 원칙** — 데이터(`data/`)와 화면(`components/`)을 분리해, 번역가·콘텐츠 담당이
코드를 건드리지 않고 `data/`만 수정할 수 있습니다. 예를 들어 번역 문구는 `data/i18n.js`,
사건 내용은 `data/timeline.js`, 시세·이슈는 `data/market.js`에서 고칩니다.
`App.jsx`는 화면 전환과 상태 관리만 담당합니다.

## 화면 구성

- **타임라인(홈)** — 돈의 역사 6시대 + 시대별 세부 사건 15개
- **사건 상세** — 사건별 심층 페이지(배경·지표·차트·교훈·오늘의 연결)
- **시장 연계** — 지수/코인 시세 + 기간별 차트(1일·1개월·1년·전체) + 변동의 역사 이슈
- **마이** — 누적 투자 초상, 갈림길 기록
- **갈림길 팝업** — 접속 시 뜨는 오늘의 선택

3개 언어(한국어·영어·일본어)를 헤더의 깃발 버튼으로 전환합니다.

## 데이터에 관한 안내 (중요)

이 저장소의 데이터는 두 종류로 명확히 구분됩니다.

### 1) 실제 데이터 (초록 "실제 데이터" 배지)
사건 상세 차트 중 **1929 대공황, 닷컴버블, 2008 리먼, 비트코인, 1997 외환위기**는
공개 기록(미 연준 FRED, 한국은행 등)에서 검증한 실제 지수·가격값입니다.
정점·저점·회복 등 핵심 변곡점은 실제 수치이며, 그 사이는 대표 시점값으로 잇습니다.

### 2) 개념도 (회색 "개념도" 배지)
튤립·로마 은화·조개껍데기 등 **신뢰할 연속 데이터가 없는 고대·중세 사건**은
"흐름의 모양"만 표현한 개념도이며, 실제 수치가 아닙니다.

### 3) 시장 연계 화면의 시세
**단기 시세(1일·1개월·1년)와 실시간 가격은 데모(샘플) 데이터입니다.**
"전체" 기간의 장기 곡선과 변동 이슈는 검증된 실제 사건 기반입니다.

#### 실시간 시세 연동 방법
`src/App.jsx`의 `QUOTES` 객체와 `MarketDemo` / `QuoteChart` 컴포넌트가 시세를 다룹니다.
실제 서비스에서는 이 부분을 시세 API(예: Finnhub, Alpha Vantage, Twelve Data 등)나
TradingView 위젯으로 교체하세요. 차트 컴포넌트는 그대로 두고
`ranges` 데이터를 API 응답으로 채우는 구조라 교체가 단순합니다.

## 법적·운영 주의사항

- **직접 매매 기능을 넣지 마세요.** 사이트 내에서 주식·코인을 직접 사고팔게 하면
  금융투자업 인가 대상이 됩니다. 본 프로젝트는 거래소·증권사로 **연결(레퍼럴)만** 합니다.
- 레퍼럴/제휴 링크는 가입·거래 시 수수료가 발생할 수 있음을 표기해야 합니다.
- 모든 페이지에 **"학습·참고용 자료, 실제 시장과 다를 수 있음"** 면책 고지가 노출됩니다.
- 가상자산 관련 콘텐츠는 변동성이 크다는 점을 함께 안내합니다.
- 역사 서술의 연도·인물·수치는 정식 발행 전 표준 사료로 교차검증을 권장합니다.
  (출처가 불분명한 일화는 "전해진다"로 표기되어 있습니다.)
- 영어·일본어 번역은 의미 전달 위주로 작성되었으며, 정식 서비스 전 원어민 감수를 권장합니다.
